# 🤖 Мобильный Антигравити (Telegram Bot)

Телеграм-бот для управления вашим ПК, файлами и AI-ассистентом Антигравити прямо с вашего телефона из любой точки мира!

---

## 🚀 Возможности

- 🧠 **Автономный AI (Gemini 2.5 Pro / Flash)**: Редактирование, создание и анализ кода с использованием автономных инструментов (Function Calling).
- 💻 **Консоль ПК с телефона**: Выполнение любых команд через `/cmd <команда>` (PowerShell/Git/npm/Python).
- 📂 **Файловая система**: Просмотр папок (`/ls`), чтение файлов (`/cat`), скачивание файлов на телефон (`/get`).
- 📥 **Загрузка файлов**: Отправка любых документов и файлов с телефона в рабочую папку на ПК.
- 📷 **Мультимодальность**: Анализ скриншотов и фотографий.
- 🎙️ **Голосовое управление**: Голосовой ввод команд и вопросов.
- 📊 **Мониторинг ПК**: Проверка загрузки CPU, RAM и диска по команде `/status`.
- 🔒 **Максимальная безопасность**: Бот реагирует **только** на ваш конкретный Telegram ID.

## 🛠️ Пошаговая инструкция по настройке

### 1️⃣ Получение токенов (занимает 2 минуты)

1. **Telegram Bot Token**:
   - Откройте Telegram и найдите официального бота **`@BotFather`**.
   - Напишите команду `/newbot` и следуйте инструкциям (задайте имя и username бота).
   - Скопируйте полученный **Token** (выглядит как `123456789:ABCdefGHI...`).

2. **Ваш Telegram User ID** (для безопасности):
   - Найдите в Telegram бота **`@userinfobot`**.
   - Напишите ему любое сообщение или `/start`.
   - Скопируйте ваш числовой **Id** (например: `987654321`).

3. **Gemini API Key**:
   - Перейдите на сайт [Google AI Studio](https://aistudio.google.com/).
   - Нажмите **Create API key** и скопируйте созданный ключ.

### 3️⃣ Запуск бота на ПК

В терминале в папке проекта вы можете запустить бота одной командой:

```bash
npm start
```

---

### 🔄 Автозапуск 24/7 в фоновом режиме (Опционально)

Чтобы бот работал постоянно и не закрывался при закрытии окна консоли, установите менеджера процессов `pm2`:

```bash
cmd /c npm install -g pm2
cmd /c pm2 start src/bot.js --name "antigravity-bot"
```

Для проверки статуса фонового бота:
```bash
cmd /c pm2 status
```

## 📱 Список команд в Telegram

| Команда | Описание |
|---|---|
| Текст / Вопрос | Общение с AI, написание и редактирование кода на ПК |
| `/cmd <команда>` | Выполнить консольную команду (например `/cmd git status`) |
| `/ls [папка]` | Просмотреть список файлов в директории |
| `/cat <файл>` | Прочитать содержимое файла |
| `/get <файл>` | Скачать файл с ПК на мобильный телефон |
| `/status` | Проверить состояние ПК (CPU, ОЗУ, Диск) |
| `/model [flash\|pro]` | Сменить модель AI |
| `/clear` | Сбросить историю диалога |
| 📷 Фото / 🎙️ Голос | Отправить фото или голосовое сообщение |
